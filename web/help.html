Seção de ajuda do Codegen

Introdução ao Codegen

O Haftware Codegen é um sistema concebido com o intuito de facilitar e agilizar a criação de softwares que, em muitos casos, possuem muito código repetitivo, que é exatamente igual para todas as dezenas de classes presentes no modelo de dados desse sistema.

O Codegen pode, com um pouco de configuração, gerar completamente:

  • Data Access Objects (DAOs)
  • CRUDs para as tabelas
  • Forms de aceitação em HTML ou XHTML
  • Endpoint backend REST para GET, POST, PUT... em XML ou JSON
  • Front-end consumidor de serviços REST para GET, POST, PUT... em XML ou JSON
  • Validação Frontend com JavaScript
  • Validação Backend em qualquer linguagem
  • Queries SQL para criação de tabelas ou consultas, inserts e outras operações no banco
